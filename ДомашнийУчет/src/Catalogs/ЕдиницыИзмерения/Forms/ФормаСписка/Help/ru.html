Добавление новой единицы измерения
  • В списке единиц измерения нажмите на кнопку Создать.
  • Будет открыта карточка новой единицы измерения.
  • В карточке единицы измерения заполните данные о коде единицы измерения, полное и краткое наименование, международное сокращение, тип единицы измерения.
  • При заполнении данных используйте данные Общероссийского классификатора единиц измерения.
  • После ввода данных нажмите на кнопку Записать и закрыть.


Для ввода новой единицы измерения в соответствии с классификатором используйте кнопку Подобрать из классификатора.
